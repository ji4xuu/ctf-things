# Forensic

This directory contains basic forensic challenges and solutions.

## Topics Covered

- File Analysis
- Metadata Extraction
- Steganography
- Memory Forensics
- Network Forensics (PCAP Analysis)
- Disk Forensics
- Log Analysis
- Data Recovery
- File Carving

- [Digital Forensics - SANS](https://www.sans.org/blog/list-of-resource-links-from-forensics-courses/) - Forensics resources
- [Autopsy](https://www.autopsy.com/) - Digital forensics platform
- [Volatility](https://www.volatilityfoundation.org/) - Memory forensics framework
- [Wireshark Tutorial](https://www.wireshark.org/docs/) - Network analysis

## Tools

- Wireshark (Network analysis)
- Volatility (Memory forensics)
- Autopsy (Disk forensics)
- Binwalk (Firmware analysis)
- ExifTool (Metadata extraction)
- Steghide/stegsolve (Steganography)
- strings, xxd, hexdump (File analysis)
