# Reverse Engineering

This directory contains basic reverse engineering challenges and solutions.

## Topics Covered

- Binary Analysis
- Assembly Language (x86, x64, ARM)
- Debuggers (GDB, x64dbg, IDA)
- Disassemblers
- Static Analysis
- Dynamic Analysis
- Decompilation
- Malware Analysis Basics
