# Pwn

This directory contains basic exploitation (pwn) challenges and solutions.

## Topics Covered

- Buffer Overflows
- Stack Exploitation
- Heap Exploitation
- Return-Oriented Programming (ROP)
- Format String Vulnerabilities
- Shellcode Writing
- Binary Protections (ASLR, DEP, PIE, Stack Canaries)
- Return-to-libc

## Resources

- [pwn.college](https://pwn.college/) - Binary exploitation course
- [ROP Emporium](https://ropemporium.com/) - ROP challenges
- [Nightmare](https://guyinatuxedo.github.io/) - Binary exploitation tutorial
- [how2heap](https://github.com/shellphish/how2heap) - Heap exploitation techniques

## Tools

- pwntools (Python library)
- GDB with GEF/PEDA/pwndbg
- ROPgadget
- one_gadget
- checksec
